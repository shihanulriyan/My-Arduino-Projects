## 🏠 Smart Motion Detection Security System

### 📷 Project Model

![Smart Motion Detection Security System](WhatsApp%20Image%202026-07-19%20at%2020.04.43.jpeg)

### 📖 Description

An Arduino-based security system that detects motion using a PIR sensor. When motion is detected, the LED and buzzer are activated to provide an alert.

### 🛠 Components

- Arduino Uno
- PIR Motion Sensor
- LED
- Active Buzzer
- 220Ω Resistor
- Jumper Wires
